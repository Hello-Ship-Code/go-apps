### 📦 Getting Started  
1. Clone the repo:  
   ```bash
   git clone https://github.com/Hello-ship-code/go-projects.git
   ```  
2. Navigate into the project folder:  
   ```bash
   cd go-projects
   ```  
3. Run the project:  
   ```bash
   go run main.go
   ```
